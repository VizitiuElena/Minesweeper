Instructiuni de rulare:
1. Descarcare freeglut: http://freeglut.sourceforge.net/index.php#download
2. Copierea folderului include din freeglut in folderul include din C:\Program Files\CodeBlocks\MinGW\include\GL
3. Copierea folderului lib din freeglut in folderul lib din C:\Program Files\CodeBlocks\MinGW\lib
4. Copierea fisierului freeglut.dll unde este executabilul aplicatiei
5. In Codeblocks la settings->compiler->linker settings adaugare libfreeglut.a si libopengl32.a
